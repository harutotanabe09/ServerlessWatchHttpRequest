## Getting Started

Serverless framework install

### Prerequisites

install serverless framework

```sh
npm install -g serverless
```

Set Key

```sh
export AWS_ACCESS_KEY_ID=XXXXX
export AWS_SECRET_ACCESS_KEY=XXXXX
```

### Deploy

```sh
serverless deploy -v
```

### Init Settig

```sh
sls create --template aws-nodejs --path my-service
```
